Welcome to the Obscuro Gateway

Three clicks to setup encrypted communication between MetaMask and the Obscuro Network. Upon hitting Begin, you'll be asked to:
  1. Add the Obscuro Testnet network as a new network to MetaMask. Check the Network URL base is https://testnet.obscu.ro and the chain ID is 443 and hit Approve.
  2. Allow MetaMask to switch networks to the Obscuro Testnet network.
  3. Sign the Signature Request (this is not a transaction) to register a unique viewing key tied to your wallet account that allows only you to view your transactions. You will get a request for each of your accounts, only hit Sign on the accounts you want to connect.
